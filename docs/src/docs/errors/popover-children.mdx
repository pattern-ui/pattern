---
title: 'Popover component error'
error: 'Popover.Target component children should be an element or a component that accepts ref, fragments, strings, numbers and other primitive values are not supported'
---

## Error source

[Popover.Target](/core/popover/) was used with invalid children:

```tsx
import { Popover } from '@pattern/core';

function DemoWithError() {
  return (
    <>
      <Popover.Target label="Throws">Raw string, NOT OK – will throw error</Popover.Target>
      <Popover.Target label="Throws">{2}</Popover.Target>

      <Popover.Target label="Throws">
        <>Fragment, NOT OK, will throw error</>
      </Popover.Target>

      <Popover.Target label="Throws">
        <div>More that one node</div>
        <div>NOT OK, will throw error</div>
      </Popover.Target>
    </>
  );
}
```

## How to fix

Popover.Target children should be either an element or component that supports ref:

```tsx
import { Popover, Badge } from '@pattern/core';

function Demo() {
  return (
    <>
      <Popover.Target label="OK">
        <button>Native button – ok</button>
      </Popover.Target>

      <Popover.Target label="OK">
        <Badge>Pattern component – ok</Badge>
      </Popover.Target>
    </>
  );
}
```
