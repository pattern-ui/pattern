## Installation

Package depends on @pattern/core and @pattern/hooks.

Install with npm:

```sh
npm install @pattern/nprogress @pattern/core @pattern/hooks
```

Install with yarn:

```sh
yarn add @pattern/nprogress @pattern/core @pattern/hooks
```

## Setup NProgress

Add `NProgress` to your app:

```tsx
import { PatternProvider } from '@pattern/core';
import { NProgress } from '@pattern/nprogress';

function App() {
  return (
    <PatternProvider>
      <NProgress />
      <YourApp />
    </PatternProvider>
  );
}
```

## Event based functions

`@pattern/nprogress` exports several functions which can be used to perform certain actions
from any part of your application:

```tsx
// All functions can be called from anywhere (not only from components)
import {
  addNProgress,
  decreaseNProgress,
  setNProgress,
  startNProgress,
  stopNProgress,
  resetNProgress,
} from '@pattern/nprogress';

// Adds progress to the current progress
addNProgress(10);

// Decreases the current progress
decreaseNProgress(10);

// Sets a custom progress
setNProgress(50);

// Starts the progress bar
startNProgress();

// Stops the progress bar
stopNProgress();

// Resets the progress bar
resetNProgress();
```
