## Usage

By default, loader will be rendered with `theme.primaryColor`.
You can choose any color defined in [theme.colors](/theming/extend-theme/#default-colors):

<ControlDemo data={LoaderDemos.control} />

## Configure default loader

You can configure default loader in [PatternProvider](/theming/pattern-provider/),
it will be used in Loader and [LoadingOverlay](/core/loading-overlay/) components by default:

```tsx
import { PatternProvider } from '@pattern/core';

function Demo() {
  return (
    <PatternProvider theme={{ loader: 'bars' }}>
      <YourApp />
    </PatternProvider>
  );
}
```

## Size

Size controls loader height or width (depends on loader variant). Loader has predefined sizes: xs, sm, md, lg, xl.
Alternatively, you can use a number to set width or height in px:

```tsx
<Loader size="sm" /> // -> predefined small size
<Loader size={50} /> // -> size in px
```
